# Audio-to-UI Pitch Deck Builder 🎙️🤖

An application that leverages Google Gemini's advanced multimodal capabilities to turn live voice descriptions directly into interactive UI components.

## How to Run Locally
1. Download or clone this repository folder workspace.
2. Create a `.env` file in the root folder and add your secret token key:
   ```text
   GEMINI_API_KEY=your_actual_key_here
   ```
3. Open your terminal in this folder and run:
   ```bash
   npm install
   ```
4. Start the backend server:
   ```bash
   node server.js
   ```
5. Double-click `index.html` to open it in your browser, click **Record**, and pitch!
